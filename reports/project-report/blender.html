Blender (Roger, 5pt)

For the final project I modeled a cave scene. The scene is based on the blenderguru tutorial, its from 2013 so there were some differences to blender 2.90 which I used. No external models were used.
The cave modeling consists of a subdivided cylinder, which is deformed using sculpting or proportional editing. Various texture maps are applied, notably the displacement map. Textures are from texture haven and poliigon.
Hair particle systems are used to place smaller objects in the cave based on some painted vertex weights. For this a group of template objects are created.

I deviated slightly from the tutorial and added some water, this is simply a sculpted plane with a wave patter. The idea was to have a blue moonlit scene in constrast to orange interior lighting. To add some sense of scale I added a wolf silhouette. For this I took a reference image from online and traced the outline, so it is a plane with a dark material.
